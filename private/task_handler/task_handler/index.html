    .on_task

    source

    .on_task(
   task: Task, message: Message
)

    The callback function, which will be called from the rabbitmq library It deserializes the amqp message and then checks, if the requested taks is registered and calls it. If the task name is on the blocklist, it will be rejected, rescheduled and then wait for some time

    Returns either None or a new task
